     6→# Agent SDK
     7→
     8→Build AI agents that work autonomously for end users. Every Polsia company can be an AI company.
     9→
    10→**Default:** Opus 4.6 with extended thinking. Polsia handles model selection and turn limits.
    11→
    12→## MANDATORY: All AI Features Must Use Polsia API
    13→
    14→**NEVER implement AI directly in company apps. Not now, not later.**
    15→
    16→This means:
    17→- NO `anthropic.messages.create()`
    18→- NO `openai.chat.completions.create()` for AI features (only allowed for OCR/embeddings)
    19→- NO direct Claude API calls
    20→- ONLY use Polsia API endpoints
    21→
    22→**Why?** Company apps are autonomous. Polsia MUST have visibility into all AI usage for:
    23→- Debugging when things go wrong
    24→- Cost tracking and billing
    25→- Prompt injection for bug reporting
    26→- Monitoring and rate limiting
    27→
